# V2-4 보더 레디어스 프롬프트

## 요청 원문

> 아래 조건으로 워킹버디 보더 레디어스 토큰을 다시 정의해줘.
>
> [절대 원칙]
> - 피그마에서 실제 사용된 값만 포함한다
>
> [피그마에서 확인된 실제 사용값 - 이것만]
> - 10px: small card
> - 12px: card
> - 9999px: pill / button
>
> [출력]
> 3개 토큰만 정의:
> - 토큰명 / 값 / 사용 컴포넌트
> - none(0)이나 미사용 단계는 포함하지 말 것

---

## V1 대비 달라진 점

| 항목 | V1 | V2 |
|---|---|---|
| 토큰 수 | 8단계 (none~full) | **3개** |
| 미사용 포함 | xs(4), sm(8), xl(16), 2xl(24), none | 없음 |
