# V2-3 스페이싱 결과물

선택: **옵션 B — 4px 그리드 근사화**

---

## 스페이싱 토큰 (4개)

| 토큰명 | 값 | 용도 |
|---|---|---|
| `space-2` | `8px`  | 아이템 row 여백, 아이콘↔텍스트 간격 |
| `space-3` | `12px` | 섹션 래퍼 vertical padding, 내비게이션 상단, 로고 gap, 상단바 아이콘 gap |
| `space-4` | `16px` | 카드 내부 vertical padding, 화면 edge padding |
| `space-5` | `20px` | 화면/카드 horizontal padding, 카드 내부 요소 위치 |

---

## 조정 이력

| 원본값 | 조정값 | 차이 | 영향 위치 |
|---|---|---|---|
| `10px` | `12px` | +2px | 카드 섹션 래퍼 상하 여백 |
| `13px` | `12px` | -1px | 상단바 아이콘 그룹 gap |
| `22px` | `20px` | -2px | 버디 보내기 카드 내부 레이블 위치 |
